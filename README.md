# Home Assistant Add-on Repository

## Add-ons

### [Simple NAS](./simple_nas)

Verwandelt deinen Home Assistant Server in einen vollwertigen Netzwerkspeicher (NAS) mit Samba-Freigaben und Web-Oberfläche.

## Installation

1. In Home Assistant: **Einstellungen → Add-ons → Add-on Store**
2. Oben rechts **⋮ → Repositories**
3. Diese URL eintragen:
   ```
   https://github.com/gregorwolf1973/EasyNas-HASSIO-Addon
   ```
4. **Simple NAS** erscheint im Store → Installieren → Starten
